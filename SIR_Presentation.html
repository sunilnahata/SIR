Rules of the game

Player then chooses: - No switch - Switch - Random choice

Goal: maximize expected winnings

Why this is interesting

  • Intuition: “Two boxes left → 50/50”
  • Reality: conditional probability changes the situation
  • We’ll test by simulation (1000 runs per strategy)

Simulation

We simulate three functions (no switch, switch, random). Each game:

  1. Create boxes c(1e6, 0.10, 0.10) placed randomly
  2. Player makes an initial uniform random pick
  3. Host opens a dime box among the remaining
  4. Player applies a strategy to pick final box
  5. Repeat N times and compute mean winnings

No Switch

# One game: No Switch
game_no_switch <- function() {
  boxes <- c(1e6, 0.10, 0.10)
  pick <- sample(1:3, 1)
  return(boxes[pick])
}

Always switch

# One game: Always switch
game_switch <- function() {
  boxes <- c(1e6, 0.10, 0.10)
  pick <- sample(1:3, 1)
  remaining <- setdiff(1:3, pick)
  presenter <- remaining[boxes[remaining] == 0.10][1]
  final_choice <- setdiff(remaining, presenter)
  return(boxes[final_choice])
}

Random choice

# Random choice
game_random <- function() {
  boxes <- c(1e6, 0.10, 0.10)
  pick <- sample(1:3, 1)
  remaining <- setdiff(1:3, pick)
  presenter <- remaining[boxes[remaining] == 0.10][1]
  final_choice <- sample(c(pick, setdiff(remaining, presenter)), 1)
  return(boxes[final_choice])
}

1000 simulations per case

# Repeat 1000 games
N <- 1000

mean_no_switch <- mean(replicate(N, game_no_switch()))
mean_random    <- mean(replicate(N, game_random()))
mean_switch    <- mean(replicate(N, game_switch()))

means <- c("No Switch" = mean_no_switch,
              "Random" = mean_random,
              "Switch" = mean_switch)

Results — Mean Winnings

Interpretation

  • No switch ≈ 33%

  • Random ≈ 50%

  • Switch ≈ 66% ← Best strategy

  • Switching increases the win probability by almost two times than staying.

Why switching wins

  • Initial pick has probability 1/3 of being the million

  • The two other boxes together have 2/3

  • Host removes one losing box among the two others

  • The remaining closed box therefore carries the full 2/3 probability
